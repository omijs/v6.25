## omi-cli

### New Project by Omi

```bash
$ npx omi-cli init my-app
$ cd my-app           
$ npm start           # develop
$ npm run build       # release
```


### New Component by Omi

```bash
$ npx omi-cli init-component my-component
$ cd my-app           
$ npm start           # develop
$ npm run build       # release
```

### Update to the latest version

```bash
$ npm i omi-cli -g 
```
