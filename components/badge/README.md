## Badge

Mark in the upper right corner of the icon

* [→ CodePen](https://codepen.io/omijs/pen/WNQOdaB)

## Import

```js
import '@omiu/badge'
```

Or use script tag to ref it.


```html
<script src="https://unpkg.com/@omiu/badge"></script>
```

## Usage

```html
<o-badge></o-badge>
```

## API

### Props

```tsx
{
  dot?: boolean,
  content?: string,
  x?: number,
  y?: number
}
```

### 默认属性

```tsx
{
  x: 0,
  y: 0
)
```
