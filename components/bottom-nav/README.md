## BottomNav

Mobile bottom navigation

* [→ CodePen](https://codepen.io/omijs/pen/zYvdjEY)

## Import

```js
import '@omiu/bottom-nav'
```

Or use script tag to ref it.


```html
<script src="https://unpkg.com/@omiu/bottom-nav"></script>
```

## Usage

```html
<o-bottom-nav></o-bottom-nav>
```
