## 导入

```js
import '@omiu/loading'
```

或者直接 script 标签引入。


```html
<script src="https://unpkg.com/@omiu/loading"></script>
```

## 使用

```html
<o-loading> </o-loading>
```


## API

### 属性

```tsx
{
  size?: number,
  color?: string
}
```

### 默认属性
```tsx
{
  size: 40
)
```
