## 导入

```js
import '@omiu/avatar'
```

或者直接 script 标签引入。


```html
<script src="https://unpkg.com/@omiu/avatar"></script>
```

## 使用

```html
<o-avatar> </o-avatar>
```


## API

### 属性

```tsx
{
  alt?: string,
  src?: string,
  size?: number,
  square?: boolean,
  icon?: string,
  color?: string,
  backgroundColor?: string
}
```

### 默认属性
```tsx
{
  size: 40,
  color: 'white',
  backgroundColor: '#ccc'
)
```
