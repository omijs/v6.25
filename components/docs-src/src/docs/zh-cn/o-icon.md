## 导入

```js
import '@omiu/o-icon'
```

或者直接 script 标签引入。


```html
<script src="https://unpkg.com/@omiu/o-icon"></script>
```

## 使用

```html
<o-icon></o-icon>
```


## API

### 属性

```tsx
{
  path?: string,
  paths?: object,
  view?: number,
  scale?: number,
  color?: string,
  rotate?: boolean
}
```

### 默认属性
```tsx
{
  view: 1024,
  scale: 2
)
```
