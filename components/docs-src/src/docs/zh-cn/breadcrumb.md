## 导入

```js
import '@omiu/breadcrumb'
```

或者直接 script 标签引入。


```html
<script src="https://unpkg.com/@omiu/breadcrumb"></script>
```

## 使用

```html
<o-breadcrumb> </o-breadcrumb>
```


## API

### 属性

```tsx
{
  split: string,
  items: any[]
}
```

### 默认属性
```tsx
{
  split: '/'
)
```
