## 导入

```js
import '@omiu/popover'
```

或者直接 script 标签引入。


```html
<script src="https://unpkg.com/@omiu/popover"></script>
```

## 使用

```html
<o-popover></o-popover>
```


## API

### 属性

```tsx
{
  effect?: string,
  position?: string,
  trigger?: string
}
```

### 默认属性
```tsx
{
  effect: 'light',
  position: 'bottom',
  trigger: 'click'
}
```
