## Install

### Via script

```html
<script src="https://unpkg.com/omi"></script>
<script src="https://unpkg.com/@omiu/chart"></script>
```

### Via npm

``` bash
npm install @omiu/chart
```

Then:

```js
import '@omiu/chart'
```
