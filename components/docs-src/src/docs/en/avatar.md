## Import

```js
import '@omiu/avatar'
```

Or use script tag to ref it.


```html
<script src="https://unpkg.com/@omiu/avatar"></script>
```

## Usage

```html
<o-avatar></o-avatar>
```

## API

### Props

```tsx
{
  alt?: string,
  src?: string,
  size?: number,
  square?: boolean,
  icon?: string,
  color?: string,
  backgroundColor?: string
}
```

### defaultProps

```tsx
{
  size: 40,
  color: 'white',
  backgroundColor: '#ccc'
)
```
